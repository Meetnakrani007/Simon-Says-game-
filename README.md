# 🎮 Simon Says Game

A fun and interactive **Simon Says** game built using **HTML, CSS, and JavaScript**. Test your memory by repeating the correct sequence of colors! 🔴🟢🔵🟡

## 🚀 How to Play?
1. The game will show a sequence of flashing colors.
2. **Memorize the sequence** and repeat it by clicking the buttons in the same order.
3. Each round adds a new step, making it **progressively harder**.
4. A wrong move resets the game—try to beat your best score!

## ✨ Features
- Smooth animations for an engaging experience.
- Dynamic difficulty as the sequence grows.
- Fully responsive for mobile and desktop play.
- Built with **pure JavaScript, HTML, and CSS** (no frameworks required).

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript (ES6)  

## 📦 Installation
1. Clone the repository:  
git clone https://github.com/Meetnakrani007/Simon-Says-game-.git

2. Open `index.html` in your browser and start playing!
